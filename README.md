# Frontend Basics (HTML, CSS, JavaScript)

A simple frontend mini project to demonstrate core web fundamentals:
HTML structure, CSS styling, and JavaScript DOM interaction.

## Features
- Clean HTML structure
- Basic CSS styling
- JavaScript DOM manipulation
- Event handling (click interaction)

## Project Structure
html/
├── index.html
css/
├── style.css
js/
├── app.js

markdown
Копировать код

## How it works
- The page displays a heading and a paragraph
- CSS styles the layout and typography
- JavaScript listens for a click on the heading
- On click, the text color changes and an alert appears

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
